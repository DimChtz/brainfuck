# py-brainwords (brainwords)
A simple and easy to use Brainfuck code generator in Python.

# How to use (Hello world! Example)
```python
print(generateBFCode('Hello world!'))
```

# Installation

Simply import the single .py file to your project:

```python
import brainwords
```
